# 🧩 システム試験工程 成果物一覧


以下に、**システム開発における「システム試験（System Test）」工程の成果物一覧**を整理しました。
プロジェクト規模や開発プロセス（ウォーターフォール／アジャイル）により差異はありますが、一般的なウォーターフォール型を前提にしています。

---

## 🧩 システム試験工程 成果物一覧

### ① 試験計画関連

| 成果物名          | 内容概要                            | 作成目的                     |
| ------------- | ------------------------------- | ------------------------ |
| **システム試験計画書** | 試験の目的、範囲、体制、スケジュール、環境、合否基準などを定義 | 試験の全体方針を明確化し、関係者間で合意するため |
| **試験環境構築計画書** | システム試験用の環境構築手順、構成、データ準備方法を記載    | 試験環境を安定して再現できるようにするため    |
| **試験データ設計書**  | 試験で使用するデータの定義と作成ルールを記載          | 再現性と網羅性のあるデータ準備のため       |

---

### ② 試験設計関連

| 成果物名                      | 内容概要                          | 作成目的                  |
| ------------------------- | ----------------------------- | --------------------- |
| **システム試験項目一覧表**           | 試験対象機能、条件、入力、期待結果、試験観点を整理     | 試験範囲を明確化し、抜け漏れを防止するため |
| **試験仕様書（テストケース仕様書）**      | 個々のテストケースごとの手順・期待結果を詳細化       | 試験実施者が手順通り再現可能にするため   |
| **試験項目と要件対応表（トレーサビリティ表）** | 試験項目とシステム要件（要件定義書や設計書）との対応を記載 | 要件網羅性の担保とレビュー効率化のため   |

---

### ③ 試験実施関連

| 成果物名                   | 内容概要                       | 作成目的              |
| ---------------------- | -------------------------- | ----------------- |
| **試験実施報告書（試験結果記録書）**   | 実施日、実施者、結果（合否）、問題点、証跡などを記録 | 試験進捗・品質の可視化のため    |
| **試験証跡（ログ・スクリーンショット）** | 実施時の画面キャプチャ、ログ、結果出力など      | 実施の正当性・再現性を保証するため |
| **不具合管理票（障害報告書）**      | 不具合の内容、再現手順、原因、対応状況を記録     | 問題管理および品質向上のため    |

---

### ④ 試験結果・品質評価関連

| 成果物名                 | 内容概要                       | 作成目的                     |
| -------------------- | -------------------------- | ------------------------ |
| **不具合分析報告書**         | 発生傾向、影響範囲、根本原因分析、再発防止策を整理  | 品質課題の可視化・再発防止            |
| **試験完了報告書（試験総括報告書）** | 試験結果の総括、判定、残課題一覧、品質評価をまとめる | 次工程（運用・受入試験）への引継ぎ資料として使用 |
| **品質評価報告書**          | 試験カバレッジ、バグ密度、修正履歴などの定量評価   | システムのリリース可否判断に活用         |

---

### ⑤ 関連・補助成果物

| 成果物名                    | 内容概要                     | 作成目的         |
| ----------------------- | ------------------------ | ------------ |
| **試験スケジュール表**           | 日程、担当者、進捗を管理する表          | 試験進行の可視化     |
| **試験環境構成図**             | 試験環境（サーバ、ネットワーク、ツール）の構成図 | 環境再現性と依存関係管理 |
| **試験実施手順書（運用マニュアル暫定版）** | 試験環境起動・停止・デプロイなどの手順書     | 試験作業の安定運用支援  |
