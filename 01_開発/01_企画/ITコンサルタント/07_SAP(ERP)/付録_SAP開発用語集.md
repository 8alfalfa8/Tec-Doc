# ◆ SAP開発における基本用語

以下に **SAP開発における基本用語** を、
**① 全体・概念 → ② 業務／モジュール → ③ 開発（ABAP） → ④ 設定（カスタマイズ） → ⑤ データ／連携 → ⑥ テスト・運用**
の6分類で整理します。
※ **SAP未経験者〜SI/公共・金融案件の説明資料にそのまま使えるレベル**で定義しています。

---

## ① SAP全体・概念に関する用語

| 用語      | 意味                    |
| ------- | --------------------- |
| SAP     | 企業向けERPパッケージ          |
| ERP     | 企業基幹業務統合システム          |
| ECC     | 旧SAP ERP（SAP ERP 6.0） |
| S/4HANA | 現行SAP主力ERP（HANA前提）    |
| HANA    | インメモリDB               |
| クリーンコア  | 標準を壊さない拡張思想           |
| モジュール   | 業務領域単位の機能群            |
