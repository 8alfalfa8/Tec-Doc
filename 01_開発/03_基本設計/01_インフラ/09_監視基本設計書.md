# 1. 目的

本書は、本システムにおける監視対象、監視方式、閾値、通知方法およびエスカレーションルールを定義する。

# 2. 監視設計方針

## 2.1 基本方針

* 予兆検知重視（障害発生前検知）
* 自動通知・自動復旧優先
* 重要度に応じた通知レベル分離
* 24時間365日監視
* 監視漏れゼロ設計

---

## 2.2 監視分類

| 区分       | 内容           |
| -------- | ------------ |
| リソース監視   | CPU/メモリ/ディスク |
| プロセス監視   | Web/AP/DB    |
| ログ監視     | エラーログ        |
| 外形監視     | URL死活        |
| ネットワーク監視 | 帯域/VPN       |
| セキュリティ監視 | 不正ログイン       |
