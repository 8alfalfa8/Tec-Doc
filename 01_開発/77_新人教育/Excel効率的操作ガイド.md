## 【Level 1: 必須時短マウス操作】

### 1. **ダブルクリックの魔術**
| 場所 | 効果 | 従来の操作との比較 |
|------|------|-------------------|
| **列ヘッダーの境界線** | 列幅を自動調整 | 手動調整より10倍速い |
| **行ヘッダーの境界線** | 行高を自動調整 | 手動調整より10倍速い |

| **シート名タブ** | シート名を編集 | 右クリック→名前変更より速い |

**実演**：
```excel
# 列幅自動調整の最速方法
1. 列全体を選択（列ヘッダーをクリック）
```

## 【最重要5関数：毎日使う】

### 1. **SUM（合計）**
**最も基本的な集計関数**

```excel
=SUM(数値1, 数値2, ...)

```

**監視業務での使用例**：
```excel

```

**ショートカット**：
`Alt + =` で自動挿入！これを最優先で覚える

### 2. **AVERAGE（平均）**
**通常値・基準値の把握に必須**

```excel
=AVERAGE(数値1, 数値2, ...)
=AVERAGE(範囲)
```

**監視業務での使用例**：
```excel
=AVERAGE(C2:C31)       → 1ヶ月の平均応答時間
=AVERAGE(障害発生件数) → 平日の平均障害数
=AVERAGE(B2:B100)     → CPU使用率の平均
```

### 3. **COUNT / COUNTA（カウント）**
**データ件数の把握**

```excel
=COUNT(範囲)    → 数値が入っているセルの数
```

**監視業務での使用例**：

### 4. **IF（条件分岐）**
**「もし～ならば」の判断**

```excel
=IF(条件式, 真の場合, 偽の場合)
```

**監視業務での使用例**：

**入れ子（ネスト）**：

### 5. **COUNTIF / SUMIF（条件付き集計）**

**監視業務での使用例**：
```excel

=SUMIF(C:C, "CRITICAL", D:D)       → 致命的アラートのダウンタイム合計
=COUNTIF(E:E, "*timeout*")         → "timeout"を含むログ件数
```

## 【関数エラーと解決法】

| エラー | 意味 | 原因と対策 |
|--------|------|------------|
| `#NAME?` | 関数名の間違い | スペルミス、日本語入力ONになってないか確認 |
| `#N/A` | 値が見つからない | VLOOKUPの検索値が存在しない、範囲ミス |
| `#DIV/0!` | 0で割った | 分母が0、IFERRORで回避 |
| `#VALUE!` | データ型の間違い | 数値と文字列の混在、TEXT関数で変換 |
| `#REF!` | 参照先がない | 列・行を削除した、参照範囲を見直し |

```excel
2. 未復旧障害件数
3. レベル別障害件数
4. サーバー別障害件数（TOP3）
5. 平均復旧時間

【解答例】
=COUNTA(A2:A100)                    → 総障害件数
=COUNTIF(D2:D100, "")              → 未復旧件数
=COUNTIF(B:B, "CRITICAL")          → CRITICAL件数
=COUNTIF(B:B, "ERROR")             → ERROR件数
=COUNTIF(B:B, "WARNING")           → WARNING件数
=AVERAGE(IF(D2:D100<>"", D2:D100-A2:A100, "")) → 平均復旧時間
```
