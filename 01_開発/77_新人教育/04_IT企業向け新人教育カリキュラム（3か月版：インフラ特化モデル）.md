# ◆ IT企業向け 新人教育カリキュラム（3か月版：インフラエンジニア特化モデル1）

以下に、**IT企業向け 新人教育カリキュラム**（**3か月版：インフラエンジニア特化モデル**）を
**本番環境を扱える人材の入口**として、そのまま現場投入できる粒度で整理します。

対象想定：

* 新卒／未経験〜微経験
* クラウド／オンプレ混在案件を想定
* 金融・公共レベルの品質意識を前提

---

## ■ 全体ゴール（3か月後の到達状態）

✅ インフラ構成図を読める／書ける<br>
✅ Linuxで基本的な運用作業ができる<br>
✅ ネットワークとセキュリティの基礎を説明できる<br>
✅ クラウド（例：Amazon Web Services）の基本設計思想を理解している<br>
✅ 手順書・チェックリスト通りに安全に作業できる<br>
✅ 障害時に「一次切り分け」ができる

---

## ■ フェーズ構成（90日ロードマップ）

| フェーズ   | 期間    | 目的      |
| ------ | ----- | ------- |
| Phase1 | 1〜4週  | インフラ基礎力 |
| Phase2 | 5〜8週  | 構築・設計理解 |
| Phase3 | 9〜12週 | 実務OJT   |

---

# ◆ IT企業向け 新人教育カリキュラム（3か月版：インフラ特化モデル2）
