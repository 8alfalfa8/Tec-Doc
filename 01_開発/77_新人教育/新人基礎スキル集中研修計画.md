# **新人基礎スキル集中研修計画**

## 【研修の目的】

### 1. **業務遂行能力の確立**
- **主要目的**: インフラ・セキュリティ監視業務に必要な最低限のPC操作スキルを習得させる
- **具体目標**: 研修終了後、OJTにおいてメンター監督下で基本的な監視業務を実施できる

### 2. **セキュリティ意識の醸成**
- **主要目的**: 情報セキュリティの基本原則を理解し、実践できるようにする
- **具体目標**: セキュリティポリシーに違反する行動を取らない、問題発生時の報告フローを理解する

### 3. **効率的な作業習慣の形成**
- **主要目的**: キーボードショートカットを活用した効率的なPC操作習慣を身につける
- **具体目標**: マウスに依存しない基本操作ができる、作業時間を短縮する基本スキルを習得する

### 4. **チーム連携の基盤整備**
- **主要目的**: 報告・連絡・相談の基本を理解し、実践できるようにする
- **具体目標**: 業務上の問題を適切なタイミングで適切な人に報告できる

### **【カリキュラムの最適化ポイント】**

1. **時間配分の最適化**
   - 4時間日: 理論+実践のバランス学習
   - 2時間日: 実践と統合に集中

2. **学習内容の階層化**
   ```
   基礎 (Day1-2) → 応用 (Day3-4) → 統合 (Day5-6) → 評価 (Day7)
   ```

3. **セキュリティの繰り返し学習**
   - 毎日冒頭5分: セキュリティチェック
   - 最終日に理解度確認

4. **実務直結の演習設計**
   - 全ての演習で監視業務を想定
   - 実際の業務フローに沿ったシナリオ

### **【成果物と評価】**
- **毎日**: 小テスト（理解度確認）
- **Day 7**: 総合実技試験
- **最終成果物**: 監視レポートサンプル（Linux+Excel連携）

### **【今後、OJTでさらに強化する内容】**

1. **業務フローの理解**<br>
   　（発見 → 記録 → 報告）

2. **問い合わせ対応の基本対応パターン**<br>
   　- 電話・メールにおける定型フレーズ<br>
   　- 問い合わせログへの記入（5W1H）<br>
   　- 「確認します」といった一次対応の返答練習<br>

3. **障害報告の基本対応パターン**<br>
   　- 障害報告テンプレートの記入方法<br>
   　- 5W1Hを意識した記載練習<br>
   　- エスカレーション先および緊急連絡網の確認

このスケジュールでは、限られた時間で最大限の効果を上げるため、余分な内容を削ぎ落とし、監視業務に直結するスキルに特化しています。
